# 水稻数据分析平台

## 项目概述

这是一个基于 Streamlit 的水稻数据可视化分析平台，集成了数据预处理、探索性分析、主成分分析和机器学习预测等功能。项目通过交互式可视化技术，深入分析水稻生产中的关键因素与产量关系，为农业生产决策提供数据支持。

## 功能模块详解

### 1. 数据预处理 (`data_preprocessing.py`)
- 功能: 数据加载、质量检查、异常值处理、数据清洗
- 特点: 自动检测缺失值、重复数据，提供数据验证和基础清洗
- 输出: 清洗后的标准化数据集

### 2. PCA数据预处理 (`pca_preprocessing.py`)
- 功能: 为PCA分析准备数据，包括分类变量独热编码和数值变量标准化
- 特点: 自动识别分类变量，执行标准化处理，生成PCA-ready数据
- 输出: 标准化后的特征矩阵

### 3. 单变量分析 (`univariate_analysis.py`)
- 功能: 对品种、地区、产量等关键变量进行分布分析
- 可视化: 饼图、条形图、直方图、箱线图
- 分析: 统计描述、分布特征、异常值识别

### 4. 双变量分析 (`bivariate_analysis.py`)
- 功能: 探索品种、育苗方式、地区与产量之间的关系
- 统计检验: 方差分析(ANOVA)、T检验
- 可视化: 分组箱线图、小提琴图、相关性分析

### 5. 生长阶段分析 (`growth_stage_analysis.py`)
- 功能: 分析不同生长阶段降雨量和温度对产量的影响
- 方法: 皮尔逊相关性分析、散点图矩阵
- 输出: 关键生长期识别、适宜环境条件分析

### 6. PCA分析 (`pca_analysis.py`)
- 功能: 主成分分析，识别主要变化方向
- 可视化: 相关性热力图、碎石图、主成分散点图
- 分析: 方差解释率、特征重要性、降维效果评估

### 7. 线性回归预测 (`linear_regression_prediction.py`)
- 功能: 基于线性回归的产量预测模型
- 评估: RMSE、R²得分
- 可视化: 预测效果图、特征重要性、残差分析

### 8. 随机森林预测 (`random_forest_prediction.py`)
- 功能: 集成学习预测模型，与线性回归对比
- 特点: 参数调优、特征重要性分析、预测不确定性评估
- 可视化: 动态训练过程、模型对比、树状图展示

## 技术特点

1. 交互式可视化: 使用 Plotly 实现丰富的交互式图表
2. 模块化设计: 各功能独立成模块，便于维护和扩展
3. 数据驱动: 基于真实水稻数据集的分析和预测
4. 用户友好: Streamlit 框架提供直观的Web界面
5. 科学严谨: 集成统计检验和机器学习算法

## 安装与运行

### 环境要求
- Python 3.8+
- 依赖包详见 `requirements.txt`

### 安装步骤
1. 克隆项目到本地
2. 安装依赖：`pip install -r requirements.txt`
3. 准备数据文件：确保 `paddydataset.csv` 在指定路径
4. 运行应用：`streamlit run main_app.py`

### 使用流程
1. 从数据预处理开始，加载和清洗数据
2. 进行PCA数据预处理
3. 依次进行单变量、双变量、生长阶段分析
4. 执行PCA分析识别关键因素
5. 使用线性回归和随机森林进行产量预测

## 数据要求

项目使用水稻生产数据集，应包含以下关键字段：
- 品种信息 (Variety)
- 地区信息 (Agriblock)
- 产量数据 (Paddy yield)
- 环境因素 (降雨量、温度等)
- 管理措施 (育苗方式等)

## 应用价值

该平台为农业科研人员和生产者提供了：
- 数据驱动的决策支持
- 关键影响因素的量化分析
- 产量预测和风险评估
- 种植方案优化建议
